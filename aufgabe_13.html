Unser User soll von der register.html auf die Seite fragebogen.html geleitet werden. Passen Sie die Seite register.html entsprechend an. Hier sollen folgende Fragen beantwortet werden: - Mit welchem Betriebssystem haben Sie Erfahrungen (Linux, iOS, Windows)? - Wie gut schätzen Sie Ihre Kenntnisse in Windows ein (Profi, Poweruser, Einsteiger)? - Welche Programmiersprachen beherrschen Sie (C#, PHP, Java, Python)? - Haben Sie schon einmal eine eigene Website erstellt (Ja/Nein)? Unterhalb der Fragen soll ein Button mit "Abschicken" gedrückt werden können. Wenn das Formular abgeschickt wurde, soll in einer anderen Datei (danke.html) dem User eine Meldung in "auffälligem Design" mit der Meldung "Vielen Dank für Ihre Antworten!" angezeigt werden. Passen Sie die fragebogen.html so an, dass die Meldung auch erscheint.
